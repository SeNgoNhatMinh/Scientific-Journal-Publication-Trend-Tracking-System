# 機能要件一覧

| id | functionName | description | relatedRequirement | priority |
| --- | --- | --- | --- | --- |
| FR-001 | 外部データ収集バッチ処理 | OpenAlex等のAPIから論文メタデータを定期的に取得し、Research Corpusへ取り込む自動実行機能。※要確認：実行間隔 | BR-001, BR-004 | 高 |
| FR-002 | 論文データ正規化・重複排除機能 | 収集したデータを共通スキーマに変換し、DOIに基づき重複を統合するバックエンド処理。 | BR-002, BR-003 | 高 |
| FR-003 | 論文検索エンジン | キーワード、著者、発行年等による検索および属性絞り込み機能の提供。 | BR-009, BR-010 | 高 |
| FR-004 | 論文詳細表示画面 | 検索結果の論文メタデータ、抄録、外部原文へのリンクを表示する画面機能。 | BR-013 | 高 |
| FR-005 | トレンド分析エンジン | 年度別の論文数統計および成長速度を算出する分析バックエンド処理。 | BR-015, BR-016 | 高 |
| FR-006 | Research Gap検出処理 | 論文数が少ない成長ポテンシャルのある領域を特定・提示する分析機能。 | BR-018 | 高 |
| FR-007 | 可視化ダッシュボード | 研究トレンドの時系列グラフ、Research Gapヒートマップ等を表示するフロントエンド機能。 | BR-022, BR-023 | 高 |
| FR-008 | マイライブラリ管理機能 | ユーザーが論文を保存、フォルダ作成、管理するためのCRUD機能。 | BR-027, BR-028 | 高 |
| FR-009 | AI論文要約・支援機能 | AIによる論文の自動要約、関連資料の提示、専門用語解説を行う機能。※要確認：利用するLLM API | BR-033, BR-034, BR-035 | 高 |
| FR-010 | フォロー済みトピック通知機能 | 興味トピックに関連する新着論文が発生した際、ユーザーに通知する機能。※要確認：通知チャネル | BR-029, BR-030 | 中 |
| FR-011 | ユーザー管理機能 | ユーザーの登録・ログイン・プロファイル管理およびRBACによる権限制御機能。 | BR-038, BR-039 | 高 |
| FR-012 | 管理用ダッシュボード | システムエラー監視、ログ参照、データソース設定を行う管理者のための画面機能。 | BR-007, BR-040, BR-041 | 中 |
| FR-013 | 分析レポートエクスポート機能 | 統計データをCSVやPDF等の形式で出力する機能。※要確認：フォーマット仕様 | BR-025 | 中 |
