# 企画書スライド

## 学術研究トレンド追跡システム

- プロジェクト概要：学術研究トレンドの可視化と研究空白域の特定
- 目的：情報過多な研究領域における客観的な研究活動の支援
- 対象：学生、研究者、教員向けのトータルサポートプラットフォーム

> **Notes:** 本プロジェクトは、膨大な学術データから研究のトレンドを抽出し、データに基づく研究支援環境を提供します。

## プロジェクト概要

- 背景：論文数の急増に伴うトレンド把握の困難化と感覚的な意思決定
- 課題：従来の検索ツールにはトレンド分析機能やResearch Gap発見機能が不足
- 解決策：信頼できる学術データソース（OpenAlex, arXiv等）を統合した「Research Corpus」の構築
- 価値：研究テーマの選定効率化、未開拓領域の発見、研究活動の客観的分析
